## Iniciando Servidor

`npm start`

## Criando um projeto em react usando Vite

1. `npm create vite@latest`
2. Digite o nome do projeto
3. Escolha o react
4. Escolha o JavaScript
5. Entre no diretorio criado e digite `npm i`
6. Para rodar `npm run dev`
